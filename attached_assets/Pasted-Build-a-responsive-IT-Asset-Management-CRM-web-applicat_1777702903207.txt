Build a responsive IT Asset Management CRM web application called "Asset Desk Demo".

Purpose:
This app is for managing laptop and mobile assets and allowing end users to raise asset-related tickets.

Important:
Use demo/sample data only. Do not use any real company data.

User roles:
1. Super Admin
2. IT Agent
3. End User

Access rules:
- Super Admin can access everything.
- Super Admin can create/edit/delete users, agents, assets, tickets, categories, and reports.
- IT Agent can view assets, manage assigned tickets, update ticket status, and add ticket comments.
- End User can only raise asset issue tickets and view their own tickets.
